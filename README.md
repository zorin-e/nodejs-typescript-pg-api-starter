# Boilerplate for developing API on nodejs and pg with typescript, linter and jest

### Install dependencies
```
npm install
```

### Run dev server
```
npm run dev
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
```
npm run lint:fix
```

### Build code
```
npm run build
```

### Build code and run it
```
npm start
```
