# Boilerplate for developing API on nodejs, ts and pg

## Lint, prettier and jest included

## Let's started

### Install dependencies
```
npm install
```

### Run dev server
```
npm run dev
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
```
npm run lint:fix
```

### Build code
```
npm run build
```

### Build code and run it
```
npm start
```
